# 用 Python 编写的获取指定用户所有微博信息的脚本，可加入统计功能

> usage

* 开发环境 Python 3.6.5
* 依赖库 requests、pyquery
* 获取指定微博用户ID后传入 `init(user_id='2145291155')` 函数中，示例中所传入的是马云巴巴的ID
